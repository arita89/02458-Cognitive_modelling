    plotxy

    Class: simscape.logging.Series
    Package: simscape.logging

    Plot two series against each other

    Syntax

    h = plotxy(x,y,Name,Value)

    Description

    h = plotxy(x,y,Name,Value) plots values of the simulation series y along the y-axis, with values of the simulation series x along the x-axis. h is a cell array of handles to the resulting figures. Arguments x and y are objects (y can be a cell array of objects) of class simscape.logging.Series. Each object name must include a full identifier path to the series, starting with the workspace log variable name. The series must have the same time vectors. The remaining arguments are optional and provided as name-value pairs.

    For more information, including the descriptions of name-value pair arguments, see the simscape.logging.plotxy reference page.

    Examples

    Plot velocities of ports C and R of the Translational Spring block TS against each other, in mm/s:

    plotxy(simlog.TS.C.v.series, simlog.TS.R.v.series, 'xunits', 'mm/s', 'yunits', 'mm/s')

    Alternatives

    Use the simscape.logging.plotxy function.

    See Also

    simscape.logging.Series | simscape.logging.Series.plot
